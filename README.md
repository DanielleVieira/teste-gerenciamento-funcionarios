# teste-gerenciamento-funcionarios

Projeto em Java feito para um teste prático de programação.

O sistema cadastra os funcionários informados no enunciado e executa as operações pedidas, como remoção, aumento salarial, agrupamento por função, ordenação e cálculo de salários.

Para rodar:

```bash
javac -d out $(find src -name '*.java' | sort)
java -cp out Main
```
